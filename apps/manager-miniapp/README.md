# Manager Mini-program

AppID: `wx71c7c2f4bcff4434`

## MVP Screens

- group-scoped workbench and metrics
- pending, completed and cancelled orders
- order detail and optional cancellation note
- basic activity editing
- schedule editing and pause controls
- customer lookup and frozen-state handling

The manager mini-program does not initiate customer payments and does not need
to be associated with the WeChat Pay merchant account.
